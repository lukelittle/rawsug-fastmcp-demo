# MCP Slideshow - GitHub Pages

This folder contains the HTML slideshow presentation about FastMCP and the Vinyl Collection Chatbot.

## Viewing the Slideshow

- **Local**: Open `index.html` in your browser
- **GitHub Pages**: Once enabled, visit `https://lukelittle.github.io/serverless-mcp-chatbot-demo/`

## Enabling GitHub Pages

1. Go to your GitHub repository settings
2. Navigate to **Pages** in the left sidebar
3. Under **Source**, select `main` branch and `/docs` folder
4. Click **Save**
5. GitHub will build and deploy the site (usually takes 1-2 minutes)
6. Your slideshow will be available at: `https://lukelittle.github.io/serverless-mcp-chatbot-demo/`

## Features

- 16 comprehensive slides covering MCP, FastMCP, and the Vinyl Collection Chatbot
- Dark theme matching lukelittle.com
- Keyboard navigation (arrow keys ← →)
- Click navigation with Previous/Next buttons
- Progress bar and slide counter
- Fully responsive design
- Single standalone HTML file (no external dependencies)

## Navigation

- **Arrow Keys**: ← Previous slide, → Next slide
- **Buttons**: Click Previous/Next at the bottom
- **Progress**: Top bar shows progress through presentation
- **Counter**: Top-right shows current slide number

Perfect for presenting at AWS meetups or conferences!
